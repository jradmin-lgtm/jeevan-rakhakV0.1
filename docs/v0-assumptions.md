# Version 0 Assumptions (Hyderabad Pilot)

## Market Scope

- Launch city: Hyderabad.
- Operational window: 24x7 with soft SLA in pilot.
- Rollout zones: begin with high-density hospital corridors, then expand ring-by-ring.

## Language and Accessibility

- App language in V0: English + Telugu.
- Voice-first support number visible on all emergency states.
- Keep core user actions to 3 taps maximum from home screen.

## Emergency Categories (V0)

- Accident / trauma
- Cardiac emergency
- Breathing distress
- Pregnancy / neonatal
- General critical transfer

## Service Boundaries

- Android-first launch.
- Cash and digital collection both enabled.
- Manual dispatch override available in admin for mission-critical fallback.
